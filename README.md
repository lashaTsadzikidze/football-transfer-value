# ⚽ Football Transfer Value Prediction

მანქანური სწავლების პროექტი, რომელიც პროგნოზირებს ფეხბურთელების საბაზრო ღირებულებას სტატისტიკური მონაცემების მიხედვით.

## 🎯 კვლევითი კითხვა

**რა განაპირობებს ფეხბურთელის სატრანსფერო საბაზრო ღირებულებას და რამდენად შესაძლებელია მისი ახსნა სტატისტიკური ეფექტიანობით?**

### მთავარი დასკვნა

სტატისტიკური ეფექტიანობა ხსნის საბაზრო ღირებულების **56%-ს (R2 = 0.569)**. დარჩენილი **44%** განპირობებულია ამ dataset-ის გარეთ დარჩენილი ფაქტორებით: ბრენდის ღირებულება, კონტრაქტის დეტალები, კლუბის ფინანსები და სატრანსფერო ბაზრის დინამიკა.

## 🔬 მეთოდოლოგია

### 1. Explaratory Data Analysis (02_eda.ipynb)

**სამიზნე ცვლადის განაწილება:**
- მარჯვნივ-გადახრილი: €50K - €180M
- მედიანა: €6M
- log-ტრანსფორმაცია გამოყენებული მოდელირებისთვის

**მთავარი აღმოჩენები:**
- თავდამსხმელი მოთამაშეები ფასდებიან 2.7-ჯერ უფრო მეტად ვიდრე მეკარეები
- Premier League: 3-4-ჯერ უფრო მაღალი მედიანური ღირებულება სხვა Top 4 ლიგებთან შედარებით
- ასაკ-ღირებულების კავშირი არაწრფივია (პიკი 23-28 წლებში)

### 2. Feature Engineering (03_modeling.ipynb)

შექმნილი ახალი feature-ები:
- `G_A` = გოლები + ასისტები
- `xG_xAG` = Expected Goals + Expected Assists
- `Age_squared` = Age^2
- `Age_from_peak` = |Age - 23|
- One-hot encoding: პოზიცია, ლიგა, ფეხი

### 3. მოდელების შედარება

| მოდელი | R2 | RMSE (M) | MAE (M) | MAPE (%) |
|--------|-----|-----------|----------|----------|
| **Random Forest** | **0.569** | 15.71 | 7.67 | 119.8 |
| LightGBM | 0.562 | 15.73 | 7.68 | 119.5 |
| XGBoost | 0.555 | 15.70 | 7.61 | 117.8 |
| ElasticNet | 0.462 | 19.13 | 8.63 | 152.1 |

**საუკეთესო მოდელი:** Random Forest (R2 = 0.569)

### 4. SHAP ინტერპრეტაცია (04_shap.ipynb)

**ტოპ-5 ყველაზე მნიშვნელოვანი Feature-ები:**

| Feature | მნიშვნელობა |
|---------|------------|
| Progressive Passes (PrgP) | 0.352 |
| Age (ასაკი) | 0.328 |
| Expected Goals + Assists (xG+xAG) | 0.304 |
| League (Premier League) | 0.275 |
| Progressive Carries (PrgC) | 0.177 |

**ასაკის გავლენა (არაწრფივი):**
- ახალგაზრდა მოთამაშეები (<22 წელი): SHAP +0.4 - +0.7 (ფასდება პოტენციალის მიხედვით)
- პიკი ასაკი (23-28 წელი): SHAP +0.2 - +0.3 (სტაბილური ღირებულება)
- ხანდაზმული მოთამაშეები (30+): SHAP -0.3 - -1.5 (მკვეთრი ვარდნა)

**ლიგის გავლენა:**
- Premier League: +27.5% ღირებულების ზრდა დამოუკიდებლად სტატისტიკისგან
- ასახავს უფრო მაღალ კომერციულ შემოსავალს და გლობალურ ხილვადობას

## 📈 მთავარი დასკვნები

### რა განაპირობებს მოთამაშის ღირებულებას?

1. **პლეიმეკინგის უნარი (35.2%)** - Progressive passes დომინირებს
   - თანამედროვე ფეხბურთი აფასებს შემოქმედებით ნახევარმცველებს (Midfielders)
   
2. **ასაკი (32.8%)** - არაწრფივი, პიკი 23-28 წლებში
   - ახალგაზრდა მოთამაშეები ფასდება მომავალი პოტენციალის გამო
   
3. **თავდასხმითი ეფექტიანობა (30.4%)** - Expected stats უფრო მნიშვნელოვანი ვიდრე რეალური ეფექტი
   - xG/xAG უკეთესი საწინასწარმეტყველო მონაცემებია ვიდრე რეალური გოლები
   
4. **ლიგის კონტექსტი (27.5%)** - Premier League
   - დამოუკიდებელი სტატისტიკისგან
   
5. **დაცვითი წვლილი (12.0%)** - ნაკლებად ფასდება
   - tackles/blocks სუსტი მონაცემებია

### ბაზრის ირაციონალურობა

44% საბაზრო ღირებულებისთვის აუხსნელი დარჩება. განპირობებულია:
- ბრენდის ღირებულება და გაყიდვადობა
- კონტრაქტის დინამიკა (წლები, ხელფასი)
- კლუბის ფინანსური ძალა
- სატრანსფერო ბაზრის hype
- მომავალი პოტენციალი (სკაუტის შეფასება)
- პოზიციური იშვიათობა
